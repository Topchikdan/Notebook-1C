# Справочник контактов 1С (Contacts-1C)

Минимальная конфигурация 1С со справочником контактов.

## О проекте

Конфигурация содержит единственный объект — иерархический справочник **«Контакты»**, предназначенный для хранения списка контактов (физических лиц или организаций) с возможностью группировки (папки/элементы).

Исходники выгружены штатным механизмом платформы (**Конфигурация → Выгрузить конфигурацию в файлы...**) в текстовом/XML-виде, что позволяет вести историю изменений в git без хранилища конфигурации (актуально для базовых версий 1С, где хранилище недоступно).

## Состав конфигурации

### Справочники

| Справочник | Назначение |
|---|---|
| **Контакты** | Иерархический справочник контактов (поддерживает группы и элементы) |

**Реквизиты справочника «Контакты»:**

| Реквизит | Тип | Назначение |
|---|---|---|
| **Телефон** | Строка (до 20 символов) | Контактный телефон |

## Требования

- Платформа 1С:Предприятие 8.3 (для открытия и редактирования конфигурации в Конфигураторе)

## Как обновить репозиторий после изменений в конфигурации

1. Откройте базу в режиме **Конфигуратор**.
2. Выполните **Конфигурация → Выгрузить конфигурацию в файлы...**, указав эту же папку.
3. Зафиксируйте изменения:

```bash
git add .
git commit -m "Описание изменений"
git push
```

## Как развернуть конфигурацию из репозитория

1. Создайте новую информационную базу в 1С (или используйте существующую).
2. В Конфигураторе выполните **Конфигурация → Загрузить конфигурацию из файлов...**, указав путь к этой папке.
3. Обновите конфигурацию базы данных (**F7**).
